# Glossary - Marketing Cloud on Core

## A-E

**Campaign**
Standard Salesforce object used to track marketing initiatives and ROI.

**Content Builder**
Tool for creating and managing email templates and reusable content blocks.

**Email Send**
A record of an email sent to a recipient, tracked in Salesforce.

**Engagement**
Measure of how recipients interact with emails (opens, clicks, conversions).

## F-J

**Journey Builder**
Visual automation tool for creating multi-step customer journeys based on triggers and criteria.

## K-O

**List View**
Standard Salesforce feature used for segmentation in MC on Core (replaces data extensions from legacy MC).

**Marketing Cloud on Core**
Native Salesforce marketing automation solution built on the Lightning platform.

## P-T

**Permission Set**
Standard Salesforce security feature used to grant marketing capabilities to users.

**Send Definition**
Configuration that defines the audience, content, and timing for an email send.

**Subscriber (Legacy)**
Contact record in legacy Marketing Cloud - replaced by standard Leads/Contacts in MC on Core.

## U-Z

**Unsubscribe**
Opt-out preference tracked on Contact/Lead records in MC on Core.

## Legacy Marketing Cloud Terms → MC on Core Equivalents

| Legacy MC Term | MC on Core Equivalent |
|----------------|----------------------|
| Subscriber | Lead or Contact |
| Data Extension | Custom Object or List View |
| Subscriber List | List View or Report |
| Automation Studio | Flow + Journey Builder |
| Marketing Cloud Connect | Native (built-in) |
| Subscriber Key | Contact/Lead ID |
