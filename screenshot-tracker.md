## Instructions

1. Take screenshots as you work through each lab
2. Crop to relevant area
3. Add annotations (arrows, highlights) as needed
4. Save as JPG with exact filename from table above
5. Place in `/screenshots/` directory
6. Update status column: ❌ → ⏳ → ✅
7. Refresh HTML to see screenshot appear

## Screenshot Guidelines

- **Format**: JPG (for smaller file sizes)
- **Resolution**: 1920x1080 or higher (crop to relevant section)
- **File Size**: Aim for < 500KB per screenshot
- **Annotations**: Use built-in screenshot tools or image editor
  - Red arrows to highlight important elements
  - Yellow highlights for key fields
  - Clear, readable text
- **Naming**: Exact match to filename in table (case-sensitive)
