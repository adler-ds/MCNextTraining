# Module 1: Getting Started with Marketing Cloud on Core

## Learning Objectives
By the end of this module, you will be able to:
- Navigate the Marketing Cloud on Core interface
- Understand the relationship to standard Salesforce objects
- Access email and journey capabilities
- Configure basic settings

## Topics Covered

### 1.1 Accessing Marketing Cloud on Core
- App Launcher navigation
- Marketing app overview
- Understanding the Lightning interface

### 1.2 Data Foundation
- How MC on Core uses Leads and Contacts
- Difference from legacy subscriber model
- Data quality considerations

### 1.3 Essential Settings
- User permissions setup
- Email sending configuration
- Sender authentication (SPF, DKIM)
- Reply-to and from addresses

### 1.4 Quick Tour
- Email Studio overview
- Journey Builder interface
- Reporting capabilities
- Content management

## Hands-On Activities
1. Navigate to Marketing Cloud on Core in your org
2. Review permission sets
3. Explore the email templates
4. Create a test list view for segmentation

## Key Takeaways
- MC on Core is a native Salesforce app
- Uses standard objects, not subscribers
- Accessed through standard Salesforce login
- Follows familiar Salesforce patterns
